About

toMOTko was born of need for a tool that could run on my Zaurus C-1000 PDA and help me to learn/acquire new vocabulary in Japanese and Spanish. At the same time, I was very curious about Qt library and Qtopia environment and had the desire to do some C++ programming, something I hadn't done for quite a long time.

The name of the application is pronounced "to-MO-ko". It's a common first name for a Japanese woman. I chose it thinking it would give a more human and nice/cute aspect to my application. So, a user could imagine that Tomoko Sensei is teaching him/her some new vocabulary. I altered the name slightly to make it special and unique. Hence, MOT means "word" in French (my mother tongue).

The application is very simple. It asks words repetitively in a way similar to Leitner System. It allows the user to add/remove/edit words. It works on Zaurus C-1000 PDA. It should work on similar models as well as any computers with ARM-based processor using Qt/Qtopia environment. The development is made on an old Pentium-III laptop running Linux and using QT SDK for cross-compiling.

For the moment, it only supports Japanese-English cards with only one deck of cards. But I intend to support other languages (especially Spanish) and provide functionality to handle more than one deck/group of cards/words.

Features to implement (ordered by priority) :

  • Support more than one deck of cards.
  • Keep/restore state when quitting/starting the application.
  • Allow to be asked word from Japanese to English and a mixte mode.
  • Support for other languages (minimally Spanish and French).
