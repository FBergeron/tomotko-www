A proposito de toMOTko

toMOTko es una aplicación útil para aprender/memorisar nuevo vocabulario de un idioma estranjero.

Desde la versión 0.11.1, toMOTko puede funccionar sobre Windows también. Normalement, debería funccionar sobre Linux y MacOS X dentro de los proximos meses.

Desde la versión 0.6.0, es posible de aprendar varios idiomas con toMOTko. Por defecto, el inglés y el japonés estan seleccionados pero otros idiomas son disponibles.

Estaba cansado de fabricar cartas y de llevarlas conmigo dentro el tren o el autobus. Como tenía un Zaurus y que quería aprender la librería Qt, familiarme con el environmente Qtopia y programar en C++ (lo que no habia hecho desde hace casi 10 años), decidí escribir toMOTko!

El nombre se pronuncia "to-mo-ko". Es un nombre común para una muchacha japonesa. Originalmente, toMOTko sólo suportaba japonés y inglés. Desde la versión 0.6.0, es posible aprender otros idiomas también. Eligí este nombre para dar un aspecto más "humano" y bonito a mi aplicación. De este modo, un usuario podría imaginarse aprender nuevas palabras con ella (Tomoko). Cambie el centro del nombre por MOT que quiere decir "palabra" en francés (mi idioma natal) por fin de distinción.

La aplicación es muy sencilla. Pide unas palabras de una manera semejante a un Sistema Leitner. El algorithmo no es perfecto ya pero funciona bastante bien para facilitar el aprentisaje de palabras nuevas.

La aplicación contiene un gestionario de glosarios que permite añadir/quitar/editar grupos de palabras en una estructura hierarjica practica para seleccionar las palabras a aprender.

toMOTko funciona sobra las computadoras portables de tipo Zaurus SL-C1000 (ROM Sharp). En principio, debería funcionar también sobre los modelos similares (de la serie SL) asi que las maquinas con una arquitectura basada sobre un procesor ARM y usando Qt/Qtopia.

El desarollo de la aplicación es hecho sobre una computadora "dual-boot" usando Debian GNU/Linux (Lenny) donde instalé el SDK de Qt para "cross-compilar" y construirla, y Windows XP Profesional donde instalé el SDK de Qt 4.4.1 (open-source). Uso también Ubuntu y Fedora dentro de Virtual Box.

Aquí está la lista de algunas funcionalidades planificadas :

  • Conservar/restablecer el estado de la aplicación cuando la cerra/abrera.
  • Migrar sobre Windows CE y MacOS.
  • Introducir estadísticos varios.
  • Amejorar el algoritmo considerando los estadísticos.
  • Implementar una interface web permitiendo a los usuarios de compartir facilmente sus glosarios.
